# 共通テスト 情報 Ⅰ 解説教材（解答・解説）

本ディレクトリは、大学入学共通テスト「情報 Ⅰ」の **解答・解説教材**（Markdown）を管理します。  
本教材は、各設問の正答だけでなく、**選択肢の誤りの理由／考え方の手順／式変形の根拠**を明示し、再現可能な思考手順として提示することを目的とします。

---

## 1. 対象範囲（この教材で扱う領域）

本教材は、以下のような共通テスト情報 Ⅰ の典型分野に対応します。

- 情報と法規（著作権、個人情報、利用目的・第三者提供、セキュリティ）
- 情報デザイン（ラスタ／ベクタ、図表の読み取り）
- データの活用（表の行・列、条件指定、集計、参照）
- アルゴリズムとプログラミング（配列、添字、繰り返し、条件分岐、擬似コード）
- 情報通信ネットワーク（パスワード、認証方式、公衆無線 LAN）
- データの表現（ビット、符号化、迷路の表現、データ量比較）

---

## 2. 原稿の体裁（この教材の“書き方”）

### 2.1 見出し構造

- `# 第 n 問`
- `## 問 n`
- `### a / b ...（小問）`

例：

```md
# 第 1 問

## 問 1

### a ア ③

### 2.2 解答表示（解答番号の書き方）

- 記号：ア・イ・ウ…／オ・カ…など
- 選択肢番号：⓪ ① ② ③ ... を使用
- 正誤：選択肢解説末尾に **正** / **誤** を付ける
```
